✅ Mode tablette — 3 étapes simples
1
Gardez ce fichier ouvert sur votre tablette/iPad. Fermez cet assistant en cliquant «Terminer» ci-dessous.
2
Placez la tablette à la réception avec un petit carton d'invitation : «Avez-vous 2 minutes? Dites-nous comment s'est passée votre visite!»
3
Le patient remplit le sondage, clique «Envoyer» → son application courriel s'ouvre avec les réponses pré-remplies. Il n'a qu'à appuyer Envoyer!
💡 Conseil : Mettez la tablette en mode «Ne pas déranger» et désactivez les notifications pour une meilleure expérience patient.
📲 Mode QR Code — 5 étapes
1
Héberger le fichier sur GitHub Pages (gratuit). Allez sur github.com, créez un compte gratuit, créez un dépôt et déposez-y ce fichier HTML.
2
Activez GitHub Pages dans les paramètres du dépôt. Vous obtenez une URL du type : votrecompte.github.io/sondage
3
Allez sur qrcode-monkey.com (gratuit), collez votre URL, personnalisez les couleurs et téléchargez le QR code.
4
Imprimez le QR code et placez-le à la réception, en salle d'attente, ou sur le dessus du comptoir de paiement.
5
Le patient scanne → remplit le sondage sur son téléphone → clique «Envoyer» → les réponses vous arrivent par courriel!
