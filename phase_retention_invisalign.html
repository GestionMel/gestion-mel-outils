🎉 Félicitations — Phase de rétention

Votre traitement orthodontique est maintenant terminé. Nous sommes rendus à la phase de RÉTENTION.

Cette phase est aussi importante, sinon plus, que la phase active du traitement si vous voulez que le résultat obtenu reste stable à long terme.

Aujourd'hui votre dentiste vous a remis une coquille de rétention pour le haut et le bas. La coquille remise est très résistante dans le cadre d'une utilisation normale. Pour les patients qui font du bruxisme, la coquille pourrait percer — votre dentiste vous recommandera alors un autre type de rétention (des frais supplémentaires seront appliqués).

Recommandations pour la période de rétention
1
Le port des coquilles de rétention devra se faire toutes les nuits pendant 1 an.
2
Le port des coquilles de rétention se fera ensuite 3 à 4 fois par semaine pour toute la durée de votre vie.
⚠️ Si vous perdez ou brisez vos coquilles de rétention, il est de votre responsabilité d'appeler à la clinique dans les plus brefs délais afin de recommander une coquille. Des frais vous seront facturés pour une ou des nouvelle(s) coquille(s).
