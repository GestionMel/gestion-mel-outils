Mode d'emploi

Objectif

Ce fichier permet de comparer les encaissements enregistrés dans le logiciel de la clinique avec les transactions bancaires afin d'identifier les écarts et d'assurer un suivi financier rigoureux.

Étapes d'utilisation

1
Inscrire les encaissements quotidiens
Entrer les montants selon chaque type de paiement (argent, chèques, assurances, cartes, etc.).
2
Vérifier les montants bancaires
Reporter les encaissements réellement reçus selon les transactions bancaires dans la colonne Logiciel.
3
Comparer avec le logiciel
S'assurer que les montants inscrits dans le logiciel correspondent aux encaissements bancaires.
4
Analyser les écarts
La colonne Écart est calculée automatiquement — elle identifie les différences entre la banque et le logiciel.
5
Documenter les écarts
Inscrire les explications pour chaque différence observée dans la colonne Explication.

Points importants

Encaissements en circulation : montants enregistrés dans un mois mais encaissés dans un autre (ex : dépôt reçu en banque le lendemain).
Différences normales : certains écarts peuvent être normaux selon les délais de traitement des assurances ou des dépôts.

Erreurs fréquentes

  • Oublier d'inscrire un paiement
  • Entrer un montant dans la mauvaise colonne
  • Différence de date entre le logiciel et la banque
  • Paiement comptabilisé deux fois
  • Dépôt bancaire non enregistré

Bonnes pratiques

✓ Effectuer le balancement chaque jour
✓ Documenter les différences immédiatement
✓ Vérifier les écarts rapidement
✓ Garder une trace des explications
Résultat attendu : un balancement quotidien clair, structuré et sans écart non expliqué, permettant une gestion financière fiable de la clinique.
